iPad Pro 11-inch (3rd Gen) Tablet
Used with Apple Pencil 2 for handwritten notes, annotating papers and slides, and sketching out diagrams. Great companion to the MacBook for lectures.
EPOMAKER RT100 Keyboard favourite
Wisteria linear switches — smooth, quiet, and satisfying. Great typing experience for long writing or coding sessions.
Sony WH-1000XM5 Headphones favourite
Best noise-cancelling headphones I've owned. Essential for focusing in busy libraries or coffee shops. Class-leading ANC and excellent sound quality.
AirPods (3rd Gen) Earbuds
Daily carry for calls and walking around campus. Spatial audio is a nice touch for watching things on the go.
Apple Watch Series 5 Wearable
Primarily for fitness tracking, notifications, and Apple Pay. The always-on display on Series 5 is still solid.
HARPOON RGB Pro + Anker Vertical Mice
HARPOON wired for desktop use; Anker 2.4G wireless vertical mouse for ergonomic laptop work — reduces wrist strain during long sessions.
MAONO 192kHz USB Microphone Audio input
Condenser USB mic for presentations, recordings, and video calls. 192kHz/24-bit capture — overkill for Zoom but great for anything recorded.
EMEET 1080P Webcam Camera
A step up from the built-in camera for online interviews and video presentations. Has a decent built-in microphone as a backup.

Editor & Development

VS Code Code editor favourite
Main editor for Python, web work, and anything that isn't R. The Jupyter extension makes it a solid notebook environment too.
RStudio R IDE favourite
Still the best environment for heavy R work. The plot pane, environment viewer, and R Markdown integration are hard to replace.
IntelliJ IDEA Java IDE
Used for Java development in CPSC 210. The refactoring tools and debugger are genuinely excellent.
📊

Languages & Data Stack

Python Language favourite
Primary language for machine learning, data wrangling, NLP, and automation. pandas, numpy, scikit-learn, and PyTorch see the most use.
R Language favourite
Go-to for econometrics, statistical inference, and publication-quality visualisations via ggplot2. The tidyverse is genuinely hard to beat for data wrangling.
Julia Language
Used in ECON 408 (Computational Macroeconomics) with Prof. Jesse Perla. Great for numerical methods — fast like C, readable like Python.
Java Language
Used in CPSC 210 for the SmartMuse personal project. Solid OOP fundamentals — less likely to reach for it now than Python or R.
🌐

Browser & Productivity

Arc Primary browser favourite
Switched from Chrome and haven't looked back. Spaces keep research, school, and personal browsing separated cleanly. Command bar is fast and genuinely useful.
Google Chrome Secondary browser
Backup browser for anything that needs a Chromium profile or specific extensions that don't play well with Arc.
Safari Mobile browser
Default on iPhone and iPad for battery efficiency. Handoff between Mac and mobile is genuinely seamless.
Notion Notes & projects
Course notes, project planning, and long-form writing. Notion Calendar syncs everything into one place for scheduling.
Things 3 Task manager favourite
Best task manager I've used. Clean, fast, and gets out of the way. Quick Entry with a keyboard shortcut makes it frictionless to capture tasks.
Slack Communication
Used for team communication at UBC Rapid and other clubs. Also the default channel for most UBC course communities.
Apple Music Music
Works seamlessly across Mac, iPhone, and AirPods. The lossless audio is a nice bonus with the Sony XM5s.
